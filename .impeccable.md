## Design Context

### Users
Enterprise managers and operators at China Mobile (Shaanxi Branch). They use this dashboard to monitor AI agent performance, track AI-related news, and view invocation statistics in a corporate environment.

### Brand Personality
High-tech, data-driven, professional, and forward-looking. The interface should feel like a modern command center (科技感).

### Aesthetic Direction
Dark mode with glowing accents (cyberpunk/tech-oriented but clean enough for enterprise). High contrast, glassmorphism or sleek dark panels, neon blue/cyan accents typical of "tech" dashboards, but refined to avoid looking cheap. Highly interactive with smooth transitions.

### Design Principles
1. **Tech-Oriented but Usable**: Embrace a high-tech aesthetic (dark themes, glowing accents) without sacrificing readability.
2. **Interactive Elements**: Use hover states, smooth data loading animations, and dynamic charts to make the dashboard feel alive.
3. **Information Hierarchy**: Keep the exact same content but use varied typography, spacing, and visual weight to guide the eye from key metrics to supporting details.
4. **Vanilla First**: Prefer pure HTML/CSS/JS (vanilla) implementations without heavy frameworks, keeping the codebase simple and performant.
